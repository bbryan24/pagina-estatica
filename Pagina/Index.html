Introducción

Esta es la pagina que utilizamos para la documentacion de la practica RA3, en la cual se nos pide crear una pagina web para luego subirla a GitHub

Ej 1 y 2, Pasos para creacion de pagina

  1. Creamos archivo index.html
  2. Creamos el archivo estilos.csss
  3. Escribimos la pagina web
  4. Y hacemos los estilos

Ej 3, Creacion de repositorio

  • Creamos repositorio publico en GitHub
  • Añadimos los archivos
  • Captura del enlace
  • Y mostramos la pagina funcionando
